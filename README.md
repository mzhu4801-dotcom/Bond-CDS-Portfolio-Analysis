# Bond-CDS-Portfolio-Analysis
$100M corporate bond + CDS portfolio risk analysis system with real market data.

## Overview
- 15 corporate bonds ($70M)
- 6 CDS hedges ($30M)
- Real market data from Yahoo Finance
- Volatility, correlation, and risk metrics

## Files
- `bond_cds_analysis.ipynb` - Main analysis notebook
- `bond_cds_config.py` - Portfolio configuration
- `data_loader.py` - Market data loader

## Usage
1. Install: `pip install yfinance pandas numpy matplotlib seaborn`
2. Run: Open Jupyter and execute the notebook

## Key Results
- Average volatility: 34.94%
- Highest risk: AAL (450bps CDS spread)
- Outputs 4 CSV data files

## Technologies
Python, Pandas, NumPy, Matplotlib, yfinance
